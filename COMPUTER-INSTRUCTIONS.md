
# COMPUTER INSTRUCTIONS

Teclas (resumen breve):

- Flechas / W A S D: mover el cursor una casilla.
- Enter / Z: confirmar selección (simula clic izquierdo en la casilla del cursor).
- X / Backspace: cancelar / deseleccionar.
- Space: terminar turno.
- R: mostrar/ocultar rangos enemigos.
- Esc: abrir/cerrar menú.

Ratón:

- Botón izquierdo: seleccionar / confirmar / inspeccionar terreno (sincroniza la posición del cursor).

Sólo se usan estas teclas y el clic izquierdo actualmente. Si quieres que cambie asignaciones o añada rebinding, dímelo y lo implemento.
